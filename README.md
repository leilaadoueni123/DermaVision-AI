# DermaVision AI 🩺🤖

> AI-powered Skin Lesion Detection and Analysis Platform

## 📌 Overview

DermaVision AI is an intelligent web platform that uses Deep Learning to assist in the classification of skin lesions from dermoscopic images.

This project is designed for educational and research purposes only and does **not** replace professional medical diagnosis.

---

## 🎯 Objectives

- Detect skin lesions using Deep Learning
- Explain predictions with Grad-CAM
- Provide confidence scores
- Store prediction history
- Offer an intuitive web interface
